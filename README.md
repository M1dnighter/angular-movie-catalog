# Movie Catalog

Angular application for browsing a movie catalog with hover preview and a detailed movie page.

## Features

- Movie list
- Search by movie title
- Hover preview for movie cards
- Movie detail page
- Rating color based on value
- Component-based architecture:
  - movie-card
  - movie-preview
  - movie-detail
  - search-bar
- Skeleton loaders for images
- BEM-based SCSS styling
- Optimized using standalone components and pipes
- Preview appearance animation

## Technologies

- Angular (standalone components)
- TypeScript
- SCSS (BEM methodology)
- JSON Server (or similar REST API)
- HTML5 semantic markup
